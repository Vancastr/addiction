# Welcome to your addiction!
### This little program allows you to keep records of your addictions (or just habits if you are boring)

To start new record start the program with "create" key and give a name to your record, like this:
**python3 main.py --create [name]**

To delete your record, start program with "delete" key and enter name of record you want to delete.
Like this:
**python3 main.py --delete [name]**

To add a point of loss to the record of your addiction, start the program with "open" key and name of your record
Like this: **python3 main.py --open [name]**, then choose option 1.

To look at the record of your addicion, start the program the same as last time and choose the desired option
